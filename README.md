# A music website built using vue3 + typescript + vite + pinia
  
## 🚗   技术栈
  ###  🐰 vue3 
  ###  🐻 element plus
  ###  🐱 typescript
  ###  🐶 vite
  ###  🐷 pinia
  
  
  
  
## 🏠   实现功能
  ### 1.登录（二维码登录、手机号登录）
  ### 2.歌单推荐
  ### 3.歌曲播放（随机、循环、顺序）
  ### 4.播放歌词
  ### 5.全屏
  ### 6.响应式布局

